# Full Stack Authentication Project (HTML5, CSS3, JavaScript, Node.js, MongoDB)

This project is a **full-stack web application** that implements user authentication using a **Node.js + Express backend** and a **Tailwind-styled frontend** built with HTML5, CSS3, and JavaScript.  
The project demonstrates core concepts like API development, user registration, login, route protection, password security, and responsive UI design.
